About

JavaScript is a high level interpreted programming language. It runs on client machine in the browser. First the JavaScript file get into the client machine. And the client machine is a javascript Engine. With the help of Parser, the engine breaks data into small pieces. To convert the data easily. When Parser parse data the result showing with AST. Abstract Syntax Tree is the abbreviation of AST. Then it converted into machine code. A machine code is a low level programming language. Only the computer understand that language. And then the computer show us the result of JavaScript file.

JavaScript is single threaded. Event loop is one kind of loop that help to render the whole javascript file one by one. Event loop requires three plans.
1. Execute code.
2. Collecting and processing events.
3. Executing sub-tasks.
In the JavaScript file, there are some methods. Some works synchronously and some works asynchronously.
Those methods works synchronously, they follow the first plan. Execute code directly.
Those methods works asynchronously, they are following plan two and plan three. Those method are works in asynchronously, which methods store in the Queue. And Queue follow a rule. First in, First out. It's mean, those method store in the Queue in first, they are execute in first. By Event Loop, the Queue generate the methods which added first. And this is the plan number three.

Local Storage

1. Local Storage store data locally in the browser.
2. After closing the browser, the data don't removed from the browser. It store data in the browser memory.
3. Until you remove the data from your browser or uninstall browser, The data won't expire. No expire date.

Session Storage

1. Session Storage store data in a Session time in the browser.
2. After closing the browser, the data immediately removed from the browser.
3. Closing the browser, the data also removed from the browser memory immediately. No need to removed data dynamically. It expire immediately.
